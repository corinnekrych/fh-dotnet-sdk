Public Methods
 NameDescription
Create<T> Create a new data record with the specified datasetId and an instance of the model. The new data record will be synced to the cloud automatically.  
Delete<T> Delete the data record with the specified datasetId and model. The deletion will be applied to local data immediately and sync with cloud when possible. In case of collision, the collision will be recorded and the local change will be reverted to match the cloud entry.  
ForceSync<T> Invoke a sync loop almost immediately. It will guarantee a sync loop will run in the next 500 milliseconds (even the data model has set to stop sync - but not if StopAll is called.).  
GetInstance Get the singleton instance of the FHSyncClient  
Initialise Set the global FHSyncConfig for all the datasets. This will be used for all the dataset if no instance of FHSyncConfig is provided when managing a sync data model.  
List<T> List the data records for the specified datasetId.  
Manage<T> Manage the specified sync data model that implements the IFHSyncModel.  
Read<T> Read the data records with the specified datasetId and uid.  
Start<T> Start syncing the specified dataset with the cloud if possible  
StartAll Start syncing all local data models.  
Stop<T> Stop syncing the specified dataset with the cloud. All the changes will be saved locally only.  
StopAll Stop syncing all local data models.  
Update<T> Update the data record with the specified datasetId and model. The changes of the data record will be synced to the cloud automatically. In case of collision, the collision will be recorded and the local change will be reverted to match the cloud entry.
