FHSDK NamespaceFeedHenry .Net SDK API References
The parent name space defined by FeedHenry .Net SDK. It is defined inside the FHSDK.dll assembly, which is a Portable Class Library. The FHSDK.dll assembly can be referenced by other PCL projects.
